Hi mate, thanks for using this little code snipet !
To use it correctly, you just have to link the "cs-grid.js" and "cs-grid.css" files in your html document, as I did in the index given to you in this folder, Then, copy and paste this to your main.js

var myGrid = {
  gridWidthUnit: "px",
  gridWidth: 900,
  gridGutterWidthUnit: "px",
  gridColumnNumber: 12,
  gridGutterWidth: 20,
  gridRowUnit: "px",
  gridRowHeight: 20,
  activateGrid: true
};
setCSGrid(myGrid);


Feel free to change the values to your need :)
